-React, TailwindCSS

### Hosting
-Domain is registred at NameCheap
-Application is hosted on AWS Amplify

### Run it locally
1. Clone this repository
2. Run npm install in root folder
3. Run npm start to start the server
4. Access it on localhost:3000
